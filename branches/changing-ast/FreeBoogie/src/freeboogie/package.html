Note to developers

The code is written for Sun's Java 6 compiler. It follows the Sun Java coding conventions, but with two-spaces indents. (According to that, long lines are avoided.) At any point in time a user should be able to checkout and run ant and ant doc without seeing errors or warnings. The command ant test is allowed to fail from time to time. All the libraries in use appear as jar files under the directory lib. No generated file shall be under version control.
